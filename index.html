HTML
HTML(Hyper Text Mark Language)este un limbaj utilizat pentru crearea paginilor web cepot fiafisate intr-un browser.
CSS
CSS se ocupa in general cu aspectul si controlul grafic al elementelor din pagina ,cum ar fi :textul ,imaginile,culorile si asezarea acestora in cadrul ferestrei paginii.
PHP
PHPeste una dintre cele mai interesante tehnologii pentru site-uri si programare web ,imbina caracteristici dintre cele mai complexe cu simplitate in utilizare.
MySQL
Bazele de date relationale inteleg SQL (Structure Query Language ),un limbaj relativ simplu ,folosit pentru solicitarea datelor.
